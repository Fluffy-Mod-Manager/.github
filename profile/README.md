# Fluffy Mod Manager


Fluffy Mod Manager is a lightweight and easy-to-use mod manager designed to help you organize, install, and manage mods for your favorite games. With a user-friendly interface and robust features, Fluffy Mod Manager simplifies the modding experience for beginners and seasoned modders alike.

## Features

- **Easy Installation**: Quickly install and uninstall mods with a few clicks.
- **Mod Organization**: Automatically sort and categorize your mods.
- **Conflict Detection**: Detects mod conflicts and provides solutions.
- **Backup & Restore**: Safely backup your mods and restore them anytime.
- **Lightweight & Fast**: Minimal resource usage without sacrificing performance.
- **Cross-Platform Support**: Works on Windows, macOS, and Linux.

## Installation

### Windows
1. Download the latest release from the Releases page.
2. Extract the `.zip` file to your preferred directory.
3. Run `FluffyModManager.exe`.

### macOS / Linux
1. Download the latest release from the Releases page.
2. Extract the archive.
3. Run `FluffyModManager` using the terminal.

---

## Usage

1. Launch Fluffy Mod Manager.
2. Add your game directory under **Settings > Game Directory**.
3. Drag and drop your mods into the manager.
4. Use the **Activate** button to enable mods.
5. Run your game directly from Fluffy Mod Manager for a seamless experience.
